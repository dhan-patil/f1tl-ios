# F1TL — Formula 1 Timeline for iOS

F1TL is an iOS application built for Formula 1 enthusiasts who want to explore historical Formula 1 seasons, race-by-race standings, and championship progression over time.

The app allows users to analyze how the championship changed after each Grand Prix and view detailed historical race information in a timeline-based experience.

---

## Overview

F1TL provides historical Formula 1 data in an easy-to-navigate interface, allowing users to explore seasons, races, and standings progression throughout a championship.

For example, users can check championship standings after a specific race such as the 2021 Monaco Grand Prix and understand how the season evolved at that exact moment.

---

## Features

- Explore Formula 1 seasons and historical timelines  
- View race-by-race Formula 1 data  
- Check driver and constructor standings after any Grand Prix  
- Analyze championship progression throughout a season  
- Access detailed race information and results  
- Fast and responsive SwiftUI interface  
- Native iOS experience optimized for Apple devices

## How It Works

F1TL fetches historical Formula 1 race and standings data using REST APIs and organizes it into a structured timeline experience.

Users can:

1. Select a Formula 1 season  
2. Browse races from that season  
3. Open a specific Grand Prix  
4. View standings and points after that race  
5. Analyze how the championship evolved throughout the season

This makes it easy to revisit historical seasons and understand how title battles progressed over time.

## Installation

### Clone the Repository

bash git clone https://github.com/dhan-patil/f1tl-ios.git 

### Open the Project

1. Open Xcode  
2. Select Open Project  
3. Open the cloned F1TL folder  

### Run the Application

1. Select an iOS Simulator or physical device  
2. Press ⌘ + R

The application should build and run successfully.
